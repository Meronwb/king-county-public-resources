About This Project

Overview

Created for GEOG 458 at the University of Washington, this project visualizes the spatial distribution of public resources across King County, Washington. Our goal is to highlight where access is concentrated and where gaps in service exist, supporting conversations around spatial equity and infrastructure accessibility.

Maps

  • Dashboard (Map 1) – Interactive point map with layer toggles and popups showing name, address, and links for each resource.
  • Density Map (Map 2) – Heatmap visualizing geographic concentration of resources. Switch between resource types using the layer control.
  • Choropleth Map (Map 3) – Census tracts shaded by resource count. Hover to inspect individual tracts.

Data Sources

  • Seattle GeoData – Libraries, Community Centers, Parks
  • King County GIS Open Data – Public Health Clinics, Census Tracts
  • Washington State Geoportal

Tools & Libraries

  • Leaflet.js v1.9.4 – Interactive web mapping
  • Leaflet.markercluster v1.5.3 – Marker clustering
  • Leaflet.heat v0.2.0 – Heatmap layer
  • CARTO Basemaps – Map tiles
