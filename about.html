About This Project

This project was created for GEOG 458 at the University of Washington. Our goal is to visualize the spatial distribution of public resources across King County, Washington, and highlight areas of concentrated access as well as potential gaps in service.

Maps

  • Dashboard (Map 1) – Interactive point map showing all four resource types with layer toggles and popups containing name, address, and links.
  • Density Map (Map 2) – Heatmap visualizing the geographic concentration of resources. Switch between resource types using the layer control.
  • Choropleth Map (Map 3) – King County census tracts shaded by the number of public resources within each tract. Hover to see tract details.

Data Sources

  • Seattle GeoData – Seattle Public Libraries, Community Centers, Parks
  • King County GIS Open Data – Public Health Clinics, Census Tracts
  • Washington State Geoportal

Tools & Libraries

  • Leaflet.js – Interactive web mapping
  • Leaflet.markercluster – Marker clustering for parks layer
  • Leaflet.heat – Heatmap density visualization
  • CARTO Basemaps – Map tiles
